关于EMORO2560板子的一些参数

  • 三脚IO口最左侧接地，中间应该是VCC，右侧是可控的高低电平输入输出。
  • SERVO最左侧接地
  • ADC左侧接
  • GPP左侧接地
  • PWM 2x6接口
    左下为坐标（1,1）点，(1,1)PWM_0 (1,2)是PWM_1 (2,1)是PWM_2 (2,2)是PWM_3 ,(3,1)是PWM_4, (3,2)PWM_5。（6，1）和（5，2）接地
  • ISP右上角接地
  • JTAG左上和右上角接地I2C，最左侧接地

PWM

  • PWM全称Pulse width modulation，脉冲宽度调制。输出不同宽度的矩形波。emoro2560上的pwm频率是500Hz，周期2ms，可用来输出普通电平，比如呼吸灯。但是并不匹配常用舵机，比如TS90A，此类舵机适用周期20ms的。
  • 曾想过用IO，控制时间输出高低电平，达到pwm的目的，然而delay()和delaymicroseconds()函数似乎都不允许，或者说无法控制的那么精准。或许需要更改ATmega的参数，网上有类似教程，过于麻烦。遂改用servo，虽然鬼畜，不够顺滑，效果勉强。
  • analogwirte()

RGB 灯

  • 使用PWM控制（0-255），可做成呼吸灯，IO等其他的接口也可使用，只有高低，对应暗亮，不够灵活。三色不同组合叠加显示不同颜色。
  • 实际运行过快，建议用delay（10）延时。
  • for语句执行。

舵机

  • TS90A
  • 舵机棕色的线接，中间的是vcc，橘色的是高低电平输入输出。
  • 高低不同的占空比用来控制转动方向，50%时居中，理论应是180°，实际比这个角度大，考虑到使用SERVO控制（50-2500，1500停止）的，也没细查，索性也就2500最大，500最小。

Bluetooth

  • 默认密码0000，user: emoro2560
  • 每次开机就打开了，自动初始化。似乎串口每次接受1个（或一次）字符，所以使用遥感控制有待商榷，不排除这个可能。
  • 通过手机向其发送字符。Serival1。大多数教程都是serival，应该注意。

Raspberry摄像头

  • 有点复杂
  • 不过比起舵机还是小事，起码有代码能参考一点。
  • 本次使用的CSI接口的摄像头（一排线），区别于USB接口。
    安装时切记断电。
  • 配置什么的就不用说了，保险起见使用3B，并且系统也是老的。
  1. 首先打开配置使摄像头处于enable。
    sudo raspi-config
  2. 可以进行拍照
    raspistill -t 20000 -o image.jpg -rot 180 -w 1024 -h 768 -q 8 -t 20000 -tl 5000 -v
    -t 20000是延时20秒
    -t1 5000间隔五秒拍摄一张
    -o image.jpg是保存图片名称
    -rot 180镜头反转180°
    -w 1024 -h 768自定义宽高
    -q图像质量，似乎8质量还行
    -v运行摄像头时输出详细信息

视频流

分情况使用。首先使用（linux）arp -a和ifconfig判断本机当前的局域网地址IP。一个作为主机。这里总是默认192.168.XXX.XXX，根据情况判断。默认端口都是8080

  1. 下载源码
    git clone https://github.com/jacksonliam/mjpg-streamer.git

  2. 下载依赖包 

sudo apt-get install libjpeg8-dev
sudo apt-get install cmake
  1. 编译

有又出现warning的可能，建议sudo执行)

cd mjpg-streamer-master/mjpg-streamer-experimental
make clean all

  1. 添加摄像头，因为需要在网页视频流中启用摄像头。而摄像头仅仅可以拍照和拍摄简单视频，并不能达到实时监控的效果，另外ls /dev 没有video0。
    chmod 777 /etc/modules

  2. 改变文档权限。
    在该文档最后一行添加bcm2835-v4l2
    重启sudo halt

  3. 有mjpg和vlc两种方式，后者有些麻烦，反正没实现。这次使用前者。

    cd mjpg-streamer-master/mjpg-streamer-experimental
    ./mjpg_streamer -i "./input_raspicam.so" -o "./output_http.so -w ./www"
  4. 可改变一些参数
    i "./input_raspicam.so -fps 60 -x 320 -y 240"
    -p 80改变端口
    暂时就这么多吧
