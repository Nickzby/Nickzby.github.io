git简记

2021-01-22 学习 446 字
Git

其实我也不清楚怎么用，下面的是我找的别人的教程。

  • 首先vscode要安装git, 方便在前者自带的终端操作
  • 为避免浪费时间，建议直接git bash here直接操作，尤其是远程连接仓库的部分
  • 另外如果没有反应可以重启vscode，或清理工作树（未知）
git --version

查看版本也相当于检查有没有安装git.

  • 在Github或者Gitlab或者码云(应该也行)创建仓库repo,私人或者公开。

  • 在本地创建文件夹mycode(用于和线上绑定，方便同步)

  • 在vscode进入本地文件夹mycodedir \path\mycode,并初始化git init
  • 添加用户名和邮箱，只用注册时用的用户名和邮箱,gitlab使用的是域，注意后缀不是普通的@gitlab.com
git config --global user.name "myname"
git config --global user.email "myname@mymail.com"

参数--global不建议添加，这意味着全部仓库密钥都会用这个，我没有添加

  • 开始生成密钥，这里使用rsa, 2048位
ssh-keygen -t rsa -b 2048 -C "email@example.com"

之后出现

Generating public/private rsa key pair.
Enter file in which to save the key (/home/user/.ssh/id_rsa):

因为之前没有生成过密钥，默认放置如上位置，此时回车即可，\.ssh文件夹中将出现两种文件，id_rsa和id_rsa.pub

  • 将id_rsa.pub中的密钥复制粘贴至线上ssh设置中
  • 绑定本地文件夹和线上仓库
git remote add mycode git@github.com:myname/myrepo.git

之后的修改便可在本地文件夹操作

  • 可以做一次连接测试
ssh -T git@github.com
  • 如果本地文件需要更新，选择拉取
  • 需要上传云端，先提交保存至本地，然后推送
