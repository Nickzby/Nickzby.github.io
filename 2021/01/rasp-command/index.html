树莓派常用命令

以下记录树莓派常用的命令，因为系统原因，可以用其他linux相关命令。
因为曾经做的项目（树莓派电台）的原因，我这里记录了当时需要用的命令。

常用命令

sudo -i                #在树莓派中将一直以管理员身份运行
logout                 #退出管理员身份
sudo raspi-config      #打开树莓派设置
sudo halt              #合理关闭运行的后台，然后关机。 推荐使用
sudo reboot            #关机重启
sudo apt-get update    #检查更新软件
sudo apt-get upgrade   #检查更新软件  一套使用
cd                     #切换目录
./                     #运行程序  考虑实际，建议加上sudo 不然会提示权限不够,permission denied
sudo nano /Desktop/x.txt #打开编辑文件
lsusb                  #设备情况

声卡配置

lsusb                   #设备情况
alsamixer               #打开设置界面，可调声音，切换声卡
speaker-test -Dplughw:CARD=Device -c2 -twav

其中-D选项按设备名称plughw:CARD=Device选定USB声卡，-c2是左右双声道，-twav选定播放.wav声音文件，
如果没有指定.wav声音文件名，则播放缺省的声音。left right

录制声音

arecoed -Dplughw:CARD=Device -fcd -c2 -twav test.wav
可以使用arecord命令录制音频，然后使用aplay命令回放，-D选项按名称选定设备，-f选项设定音频格式，-c选项是声道数，-t选项是音频文件格式，详细选项设置内容可使用man查看。

播放声音

aplay -Dplughw:CARD=Device test.wav

其他采集回放方式

alsaloop -Cplughw:CARD=Device -Pplughw:CARD=ALSA,DEV=0
使用alsaloop命令将所采集音频回放，-C选项是采集设备，-P选项是回放设备. USB声卡采集，树莓派集成音频设备回放。
